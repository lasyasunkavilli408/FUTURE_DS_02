# 📊 Customer Retention & Churn Analysis

## 📌 Overview
Analyzed customer data to understand churn patterns and retention strategies.

## 🛠 Tools Used
- Python (Pandas, Matplotlib)

## 📊 Key Insights
- Customers with low tenure churn more  
- High charges lead to higher churn  
- Contract type impacts retention  

## 📈 Visualizations
- Churn Distribution  
- Tenure vs Churn  
- Monthly Charges vs Churn  
- Contract vs Churn  

## 🚀 Outcome
Identified key factors affecting customer churn and suggested retention strategies.
